1. Mini SIEM: Security Log Monitoring System
Description:

A simplified Security Information and Event Management (SIEM) system that collects logs from multiple sources, analyzes them, and detects suspicious activity. The platform ingests logs (system logs, authentication logs, network logs), stores them in a database, and provides a web dashboard to visualize alerts and security events

Features:
  • Log Collection: Collects logs from multiple sources (system logs, authentication logs, network logs)
  • Log Storage: Stores logs in a database
  • Log Analysis: Analyzes logs to detect suspicious activity
  • Alerting: Provides alerts for suspicious activity
  • Dashboard: Web dashboard to visualize alerts and security events
